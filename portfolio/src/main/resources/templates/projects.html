E-Note is a java web app for managing digital notes. Users can create, edit, and securely store notes online. Built with Servlets, JSP, and JDBC, it includes user authentication, session management, and MySQL-based storage. The app offers a clean interface for easy, paperless note-taking.

Book Shop App

The Book-Shop-App is a java web application for online book shopping. It lets users browse, search, and buy books, with secure login and order management. Admins manage the book catalog, while customers place orders easily. Built with Java EE, JSP, and Servlets, it offers a smooth and efficient shopping experience.
